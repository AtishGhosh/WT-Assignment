STRING OPERATIONS

  1. LENGTH OF A STRING
  2. CONCATENATION OF STRINGS
  3. REVERSE OF A STRING
  4. COPY A STRING
  5. COMPARE 2 STRINGS
  6. SUBSTRING

i)LENGTH OF A STRING:-

Function-Returns the length of string.

Syntax-strlen(string)

ii)CONCATENATION OF STRINGS:-

Function-Concatenates string s2 onto the end of string s1.

Syntax-strcat(s1,s2)

iii)REVERSE OF A STRINGS:-

Function-Reverse the string

Syntax-strrev(s1)

iv)COPY A STRING:-

Function-Copies string s2 into string s1.

Syntax-strcpy(s1,s2)

v)COMPARE 2 STRINGS:-

Function-Compare 2 strings.

Syntax-strncmp(s1,s2)

vi)SUB-STRING:-

Function-To find substring in main string

Syntax-strstr(main string,sub string)
